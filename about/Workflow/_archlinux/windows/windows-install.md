## 包管理
winstall 是 winget install 的网页管理工具。借助 winstall，用户不懂代码，也能轻松使用 winget，一次性安装所有应用程序。
winstall 的应用搜索继承了 winget 的问题，部分国产软件不支持中文搜索，比如 微信 WeChat「坚果云」只支持英文名称「Nutstore」。希望这一问题，可以随着国内用户群增多而被解决。

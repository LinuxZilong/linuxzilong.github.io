### Secondary DNS Zones
DNS 服务器可以部署在主要/次要拓扑中，其中次要 DNS服务器保存主要 DNS 服务器的 DNS 记录的只读副本。主服务器保存主区域文件，辅助服务器构成一个相同的辅助区域；DNS 请求分布在主服务器和辅助服务器之间。当主服务器区域文件全部或部分复制到辅助 DNS 服务器时，就会发生 DNS 区域传输。

操作系统中的线程
Thread是执行流程的一部分的活动实体。它是流程中任务的顺序流。它也被称为轻量级进程，因为它们共享公共资源。一个进程可以包含多个线程。线程用于提高应用程序的性能。每个线程都有自己的程序计数器、堆栈和寄存器组。但是单个进程的线程可能共享相同的代码和数据/文件。

关键术语：

proc
fork
join

# OCFS2
https://www.zstack.io/docs/2.3.2/pdf2/a_print_ZStack_OCFS2_SAS_Deploy_Tutorial.pdf

OCFS2（Oracle Cluster File System 2）是一个开源的并行文件系统，可用于在集群环境中共享文件系统。它支持多个节点同时访问同一文件系统，并提供了文件锁定和同步等功能。

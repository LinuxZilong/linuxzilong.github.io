## 使用 jsonnet 编写 grafana 配置
grafana 的 json 配置，完整形式复杂冗长，使用 jsonnet 可以精简配置，提升可维护性。

jsonnet 是 Google 2014 年开源的一个「数据模板语言」，是专为简化配置而设计的一门 DSL.
